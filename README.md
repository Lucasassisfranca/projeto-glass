# 🖥️ Projeto Glass

Projeto desenvolvido com o objetivo de **praticar e aprimorar habilidades em HTML e CSS**, com foco na construção e organização de layouts de páginas web.

## 📚 Sobre o projeto

O **Projeto Glass** foi criado como parte dos estudos em desenvolvimento web, explorando conceitos fundamentais de estruturação de páginas utilizando HTML e estilização com CSS.

Durante o desenvolvimento foram trabalhados conceitos como:

* Estruturação de páginas
* Containers
* Largura e altura de elementos
* Organização de layout
* Estilização com CSS
* Posicionamento de elementos

Esse projeto foi importante para consolidar a base de **Front-End**, especialmente na criação de layouts e organização visual.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
